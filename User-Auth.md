# Benutzer- und Rollenrechte

## Verkauf
- Darf sich normal anmelden
- Darf sich über den Button **„Kasse anmelden“** als Benutzer **Kasse** direkt anmelden
- Darf die Kasse benutzen
- Darf Produkte verkaufen
- Darf Mitgliedsguthaben als Rabatt verwenden
- Darf Geschenk-Gutscheine und Verzehrkarten in der Kasse einlösen
- Darf sich aus der Kasse heraus über den Header-Button **„Login“** mit einem echten Benutzer anmelden

## Manager
- Darf zusätzlich den Admin-Bereich öffnen
- Darf zusätzlich den Tab **Mitglieder** öffnen
- Darf zusätzlich Mitglieder anlegen und bearbeiten
- Darf zusätzlich Mitgliedsguthaben aufladen
- Darf zusätzlich den Tab **Produkte** öffnen
- Darf zusätzlich Produkte anlegen, bearbeiten und löschen
- Darf zusätzlich den Tab **Gutscheine** öffnen
- Darf zusätzlich **Geschenk-Gutscheine** erstellen
- Darf zusätzlich den Bereich **Gutscheine → Verwaltung** nur ansehen
- Darf zusätzlich im Bereich **Gutscheine → Verwaltung** keine Aktionen ausführen
- Darf zusätzlich den Tab **Finanzen** nur für den **Z-Bon-Bereich** nutzen
- Darf zusätzlich Z-Bons ansehen, vorbereiten und erstellen
- Darf zusätzlich Z-Bons nur mit Benutzern der Rolle **Manager**, **Admin** oder **TopAdmin** als „Erstellt von“ erfassen
- Darf zusätzlich als **Kassenprüfer** jedes Mitglied auswählen oder die Auswahl wieder entfernen

## Admin
- Darf zusätzlich **Verzehrkarten** erstellen
- Darf zusätzlich mehrere Verzehrkarten gleichzeitig erstellen
- Darf zusätzlich für Verzehrkarten automatisch Produktartikel anlegen und deren Lagerbestand erhöhen
- Darf zusätzlich Geschenk-Gutscheine in der Verwaltung bearbeiten
- Darf zusätzlich den Bereich **Gutscheine → Vereinskonto** verwenden
- Darf zusätzlich Abschöpfungen außerhalb des Z-Bon-Modals direkt buchen
- Darf zusätzlich die Tabs **Kategorien**, **Benutzer**, **Design** und **Datenpflege** öffnen
- Darf zusätzlich Benutzer anlegen und löschen

## TopAdmin
- Darf zusätzlich den initialen TopAdmin-Setup ausführen
- Darf zusätzlich Rollen an Mitglieder vergeben oder entfernen
- Darf zusätzlich alle Admin-Rechte ohne Einschränkung nutzen
- Ist wie der Benutzer **Kasse** ein ausgeblendeter Systembenutzer, wenn kein Mitgliedskonto dahinter liegt
